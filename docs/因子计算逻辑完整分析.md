# 复盘任务因子计算逻辑完整分析文档

## 一、系统架构概述

该系统是一个股票复盘分析引擎，主要包含以下核心模块：

```
┌─────────────────────────────────────────────────────────────────┐
│                        复盘任务流程                               │
├─────────────────────────────────────────────────────────────────┤
│  1. 数据获取 (Baostock)                                         │
│     └─> 日K线数据 (stock_daily_kline)                          │
│         └─> 成交额前100股票筛选                                  │
│  2. 因子计算 (factor_service.py)                                │
│     ├─> 原子因子计算                                            │
│     ├─> 中间因子计算                                            │
│     └─> 得分因子计算                                            │
│  3. 排名与得分 (review_service.py)                              │
│     ├─> 股票综合得分排序                                        │
│     ├─> 板块得分计算                                            │
│     └─> 大盘因子计算                                             │
│  4. 结果展示 (ReviewResult.vue)                                 │
└─────────────────────────────────────────────────────────────────┘
```

#### 2.4.2 表达式评估器 (`FactorCalculator._evaluate_expression`)

```python
# factor_service.py:1539-1570
def _evaluate_expression(self, expression: str, records: List[Dict]) -> List[float]:
    """评估表达式（针对多条记录）"""
    # 1. 提取表达式中的变量名
    var_names = set(re.findall(r'\b[a-zA-Z_][a-zA-Z0-9_]*\b', expression))
    # 2. 排除内置函数名
    builtins = {'abs', 'max', 'min', 'sum', 'round', ...}
    var_names = var_names - builtins
    
    # 3. 遍历每条记录计算
    for record in records:
        # 为缺失的因子添加默认值0
        record_with_defaults = dict(record)
        for var in var_names:
            if var not in record_with_defaults:
                record_with_defaults[var] = 0
        
        # 4. 使用 simpleeval 计算
        result = simpleeval.simple_eval(
            self._preprocess_expression(expression),
            names=record_with_defaults,
            functions=self.simpleeval_functions
        )
```

#### 2.4.3 支持的函数

```python
# factor_service.py:86-99
def _setup_simpleeval_functions(self):
    return {
        'ABS': abs,
        'MAX': max,
        'MIN': min,
        'SUM': sum,
        'AVG': lambda *args: sum(args) / len(args) if args else 0,
        'SQRT': lambda x: x ** 0.5 if x >= 0 else 0,
        'LOG': lambda x: np.log(x) if x > 0 else 0,
        'IF': lambda cond, true_val, false_val: true_val if cond else false_val,
        'ROUND': round,
        'POW': pow,
    }
```

### 5.2 因子树数据构建逻辑

```javascript
// ReviewResult.vue:686-839
const stockFactorTreeData = computed(() => {
  // 1. 获取所有因子定义
  const allFactorDefs = factorConfig.value.columns || []
  
  // 2. 构建因子定义映射
  const factorDefMap = {}
  for (const f of allFactorDefs) {
    factorDefMap[f.code] = f
  }
  
  // 3. 从因子配置获取表达式
  const expression = factorConfig.value.expression || ''
  const scoreFactorCodes = getFactorCodesFromExpression(expression)
  
  // 4. 为每个得分因子构建树
  for (const code of scoreFactorCodes) {
    // 构建根节点 (Level 3: 综合得分因子)
    const rootNode = {
      id: `root-${code}`,
      name: factor.name || code,
      value: getValue(code),
      expression: factor.expression,
      level: 3,
      levelClass: 'level-3',
      children: []
    }
    
    // 5. 添加直接依赖的因子 (Level 2: 中间因子)
    for (const depCode of allDeps) {
      const depNode = {
        id: `dep-${code}-${depCode}`,
        name: depFactor.name || depCode,
        value: getValue(depCode),
        level: 2,
        levelClass: 'level-2',
        children: []
      }
      
      // 6. 添加基础数据因子 (Level 1: 数据源)
      for (const [baseCode, baseFactor] of Object.entries(mergedFactorMap)) {
        if (!baseFactor.expression && baseFactor.calculation_method) {
          depNode.children.push({
            id: `base-${depCode}-${baseCode}`,
            name: baseFactor.name || baseCode,
            value: getValue(baseCode),
            level: 1,
            levelClass: 'level-1',
            children: []
          })
        }
      }
      
      rootNode.children.push(depNode)
    }
  }
})
```

## 六、完整因子计算流程图

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                           因子计算完整流程                                    │
└─────────────────────────────────────────────────────────────────────────────┘

输入: stock_codes (股票代码列表), trade_date (交易日期)

Step 1: 数据准备
├── 1.1 查询股票名称映射
├── 1.2 获取当日K线数据 (close_price, volume, turnover, pct_change)
├── 1.3 获取200天历史K线数据
└── 1.4 获取上证指数历史数据 (用于偏离值计算)

Step 2: 计算均线
├── 2.1 ma5 = Σ(近5日收盘价) / 5
├── 2.2 ma10 = Σ(近10日收盘价) / 10
└── 2.3 ma20 = Σ(近20日收盘价) / 20

Step 3: 计算原子因子 (Level 1)
├── 3.1 kline_field: 直接从K线获取
│   ├── close_price, volume, turnover, pct_change
│   ├── ma5, ma10, ma20
│   └── volume_y1, turnover_y1 (昨日数据)
│
├── 3.2 avg_* 方法: 历史平均成交额
│   ├── avg_amount_3d = Σ(近3日成交额) / 3
│   ├── avg_amount_5d = Σ(近5日成交额) / 5
│   ├── avg_amount_10d = Σ(近10日成交额) / 10
│   ├── avg_amount_20d = Σ(近20日成交额) / 20
│   ├── avg_amount_4_20d = Σ(第4-20日成交额) / 17
│   └── avg_amount_11_30d = Σ(第11-30日成交额) / 20
│
├── 3.3 rank 方法: 排名因子
│   └── amount_rank = rank(turnover, descending)
│
└── 3.4 python 方法: 偏离值因子
    ├── deviation_10d: 10日累计偏离值 (股票 vs 上证指数)
    ├── deviation_30d: 30日累计偏离值 (股票 vs 上证指数)
    └── remaining_deviation: 距离阈值的差距

Step 4: 计算中间因子 (Level 2)
├── 4.1 price_ma5_diff = close_price - ma5
├── 4.2 price_ma10_diff = close_price - ma10
└── 4.3 昨日均线: ma20_y1 = Σ(昨日往前20日收盘价) / 20

Step 5: 计算得分因子 (Level 3)
├── 5.1 factor1_rank (成交额排名得分)
│   └── 表达式: IF(amount_rank <= 10, 10, ...)
│
├── 5.2 factor2_ma (短线趋势)
│   └── 表达式: IF(close_price > ma5, 3, -1) + IF(close_price > ma10, 2, -0.5)
│
├── 5.3 factor3_vol (昨日同比)
│   └── 表达式: IF(volume >= volume_y1, 3, -1)
│
├── 5.4 factor4_burst (爆量)
│   └── 表达式: IF(avg_amount_3d >= avg_amount_4_20d, 3, -1)
│
└── 5.5 remaining_deviation (剩余偏离值)
    └── 基于股票与指数的涨跌幅偏离计算

Step 6: 计算总分
└── total_score = factor1_rank + factor2_ma + factor3_vol + factor4_burst + remaining_deviation * 0.1

Step 7: 排序输出
└── 按 total_score 降序排列

输出: DataFrame (包含所有因子值和总分)
```

```python
# factor_service.py:279-302
def find_dependencies(factor_codes, factor_map, visited=None):
    """递归查找表达式中使用的因子依赖"""
    if visited is None:
        visited = set()
    
    for fc in factor_codes:
        if fc in visited:
            continue
        visited.add(fc)
        
        if fc in factor_map:
            factor_def = factor_map[fc]
            if factor_def.expression:
                # 从表达式中提取因子代码
                vars_in_expr = re.findall(r'\b([a-zA-Z_][a-zA-Z0-9_]*)\b', factor_def.expression)
                # 排除函数名
                func_names = {'ABS', 'SQRT', 'MAX', 'MIN', 'AVG', 'SUM', ...}
                deps = {v for v in vars_in_expr if v not in func_names and v not in visited}
    
    return visited
```
